# 🎮 The DeepEffect Game — Discipline RPG

A gamified daily-routine tracker that turns building **DeepEffect** (a premium B2B video studio) and personal discipline into an RPG. Earn XP, kill your 4 "boss enemies", keep a daily combo streak, and level up from 🥚 Rookie → 👑 $5M Studio Boss.

Built as a single `index.html` — no build step, no dependencies. Progress is saved in your browser (`localStorage`).

## ▶️ Play
- **Locally:** open `index.html` in any browser.
- **Live (free):** host on GitHub Pages (see below) and play from your phone.

## How it works
- **Boss Fights** — defeat all 4 (Scroll Monster 📱, Night Goblin 🌙, Procrastinator Phantom 🐢, Chaos Imp 🌀) to keep your combo.
- **Daily Quests** — Fajr wake, 5 waqt namaz, workout, sales block, etc. Each gives XP.
- **Win Day** = 180+ XP and all 4 bosses defeated → combo +1.
- **Combo multiplier** — 3 days x1.2 🔥 · 7 days x1.5 🥉 · 30 days x2 🥈.
- **Business Levels** — tap a level when you hit the real milestone (first $2k client, first retainer, $5M run-rate).

## 🚀 Deploy to GitHub Pages
1. Create a new repo on GitHub (e.g. `deepeffect-game`).
2. Push this folder (see the commands your assistant gave you).
3. Repo → **Settings → Pages → Source: `main` branch, `/root`** → Save.
4. Open the link GitHub gives you (e.g. `https://<username>.github.io/deepeffect-game/`).

# DEEPEFFECT OPERATING SYSTEM
*The one document I follow without thinking. Updated weekly.*

**Target (Dec 2027):** $5M/year run-rate (~$417k/month). Aggressive but real — this is a *build-an-agency* goal, not a *sell-more-projects* goal.
**THE ONE METRIC:** New DeepEffect clients at **$2,000+ minimum** (one-off) → converting to **$5k+/month retainers** (the real unit).
**Key mindset shift:** I move from *editor* → *seller + creative director*. A team edits; I sell, set quality, and run the system. $5M is impossible solo and inevitable-ish with retainers + a team.
> Every block below either *signs a client* or *protects my ability to sign clients*. If an activity does neither, it gets cut.

## PHASE 0 — ESTABLISH THE FOUNDATION (do this in the first 2 weeks)
*You cannot sell a business that isn't built. Finish this BEFORE heavy outreach. ~2 weeks.*

**Step 1 — Lock the offer (1 day).** One sentence: "I help [B2B SaaS / coaches / agencies] turn raw footage into [short-form videos that get leads & authority]." Pick ONE niche. Confused offer = no $2k clients.

**Step 2 — Build packages & pricing (1 day).**
- Starter project: $2,000 (a "video sprint" — e.g. 4 short-form edits)
- Retainer: $5,000/mo (8–12 videos/mo) ← this is the goal of every deal
- Premium retainer: $12–15k/mo (full content engine) ← for later
- Never hourly. Always project/outcome.

**Step 3 — Proof assets (3–5 days). THIS is what's missing from deepeffect.net right now.**
- 3 strong case studies (before/after + result) on the website
- 3+ testimonials (ask past Fiverr clients TODAY — you have $40k of history, use it)
- A clean portfolio reel
- If you lack recent samples: create 2–3 spec edits for dream-brand styles

**Step 4 — Fix the website (2 days).** Add: portfolio, packages/pricing anchor, testimonials, clear CTA ("Book a free video audit"). A $2k buyer must see proof + price clarity.

**Step 5 — Set up the rails (1 day).**
- Simple CRM (Notion/spreadsheet): Lead → Contacted → Call → Proposal → Won
- Booking link (Calendly) + a 1-page proposal template + invoicing (Wise/Stripe)
- Rebuild Fiverr/Upwork as PREMIUM ($2k+ packages), used for inbound only

**Step 6 — Delivery SOP (1 day).** Write your editing process as repeatable steps (intake → script → edit → revisions → delivery). This is what lets you hire later. No SOP = no scaling = no $5M.

> ✅ Phase 0 done when: offer locked, packages priced, 3 case studies + 3 testimonials live, website fixed, CRM + booking ready, SOP written. THEN go full-force on §2 outreach.

## 2. CLIENT ACQUISITION ENGINE (the actual business)
**Weekly quotas (non-negotiable, tracked):**
- [ ] 50 cold outreaches (10/day x 5) to B2B brands needing video
- [ ] 5 personalized loom/video audits sent to dream clients
- [ ] 10 follow-ups to prior week's non-responders
- [ ] 3 sales calls booked
- [ ] 1 portfolio/case-study piece added or upgraded

**Where to hunt ($2k+ B2B clients, NOT Fiverr race-to-bottom):**
1. LinkedIn — B2B SaaS, agencies, coaches, founders posting video.
2. YouTube channels of B2B brands with bad editing = warm targets.
3. Cold email to marketing leads (use the loom-audit angle).
4. Fiverr/Upwork = REBUILD as premium-positioned, $2k+ packages only, used for inbound — not the main engine.

**The pitch frame:** Don't sell "editing." Sell *outcomes*: more leads, more watch time, more authority. Lead with a free 2-min audit of their existing video.

**Pricing:** Floor = $2,000. Never quote hourly. Quote project/outcome. Push toward monthly retainers ($3–5k) ASAP.

## 3. HABITS — INSTALL & ELIMINATE
**Install (stack onto existing anchors):**
- After I wake → drink water + sunlight.
- After coffee → 10 cold outreaches before anything else.
- After dinner → 10-min shutdown plan.

**Eliminate (make them hard):**
- Phone in morning → phone charges OUTSIDE bedroom.
- Doomscrolling → app timers, log out of socials on phone, use only for posting from desktop.
- Saying "I'll start Monday" → the minimum win (§0) runs every single day.

**Tracking:** Simple habit grid. Green = done. Goal = never two reds in a row.
